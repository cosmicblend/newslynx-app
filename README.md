newslynx-app
============

Code and Issue Tracking for Newslynx App


## Developing

#### First time

If not already installed

`npm install nodemon -g`

````bash
npm install
nodemon ./bin/www -e jade,js,styl,json
````

#### Normal

````
nodemon ./bin/www -e jade,js,styl,json
````

The sample article page is here <http://localhost:3000/article>
